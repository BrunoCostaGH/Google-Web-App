# Excel-Web-App

This was a Google Web App developed by me for a roleplay gaming community. It is a web portal that interacts with an excel spreadsheet.

Features:
  - Loading screen while modules are loading and identity is verified
  - Settings page to edit multiple variables depending on whitelist level
  - Queue on data entry to prevent data overwrite
